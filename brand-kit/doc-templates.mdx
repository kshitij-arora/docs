## Documentation Templates

Customize how your auto-generated documentation looks.

<Note>
  Doc templates are available on **Scale** and **Enterprise** plans.
</Note>

## Creating a Template

1. Go to **Brand Kit → Doc Templates**
2. Click **"+ Create a new template"**
3. Customize styling options
4. Save template

## Customization Options

- Header and footer design
- Font selections
- Color scheme
- Logo placement
- Layout structure
