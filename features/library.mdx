## Library Overview

The Library is your central hub for organizing all content - videos, documents, and folders.

## Navigation

Access the Library from the sidebar by clicking **"Library"** or navigate to `app.trupeer.ai/library`.

## Features

### Folders

Organize content into folders for better management:

<Steps>
  <Step title="Create Folder">
    Click **"Create a folder"** button in the Your Folders section
  </Step>
  <Step title="Name Folder">
    Enter a descriptive name for your folder
  </Step>
  <Step title="Move Content">
    Drag and drop videos/documents into folders, or use the options menu
  </Step>
</Steps>

<Note>
  Folder management is available on **Scale** and **Enterprise** plans.
</Note>

### Search

Use the search bar to find content quickly:

```
Search by:
- Video/document title
- Content keywords
- Date created
```

### Sorting & Views

<Tabs>
  <Tab title="Sort Options">
    - **Newest** - Most recently created first
    - **Oldest** - Oldest content first
    - **A-Z** - Alphabetical order
    - **Z-A** - Reverse alphabetical
  </Tab>
  <Tab title="View Modes">
    - **Grid View** - Visual thumbnails
    - **List View** - Compact list format
  </Tab>
</Tabs>

## Content Cards

Each piece of content displays:

| Element | Description |
|---------|-------------|
| **Thumbnail** | Preview image from video |
| **Title** | Content name (editable) |
| **Date** | Creation date |
| **Language** | Content language tag |
| **Options** | Menu for edit, delete, move, share |

## Content Actions

Click the **"..."** menu on any content card for:

- **Edit** - Open in video/document editor
- **Rename** - Change the title
- **Move to folder** - Organize into folders
- **Duplicate** - Create a copy
- **Share** - Generate shareable link
- **Delete** - Remove content

## Your Content Section

Below folders, the **"Your Content"** section shows all videos and documents:

- Click any thumbnail to open the editor
- Hover for quick actions
- Click **"See all"** for full library view
