## Automatic Documentation

Every video you create in Trupeer automatically generates a companion document - a step-by-step guide with screenshots and instructions.

<Info>
  Documents are generated automatically when you create a video. No extra steps required!
</Info>

## Document Structure

Generated documents include:

### Title & Introduction

- Auto-generated title based on content
- Summary introduction explaining the guide's purpose
- Highlighted key information box

### Step-by-Step Instructions

Each step includes:

<CardGroup cols={2}>
  <Card title="Step Number" icon="list-ol">
    Clear numbering (Step 1, Step 2, etc.)
  </Card>
  <Card title="Description" icon="align-left">
    Detailed instructions from your narration
  </Card>
  <Card title="Screenshots" icon="image">
    Relevant frames extracted from video
  </Card>
  <Card title="Timestamps" icon="clock">
    Reference to video timeline
  </Card>
</CardGroup>

## Accessing Documents

### From Video Editor

1. Open any video in the editor
2. Click the **"Document"** tab (next to "Video")
3. View and edit the generated documentation

### From Library

Documents appear alongside their source videos in your library with a document icon indicator.

## Editing Documents

You can customize generated documents:

<Steps>
  <Step title="Edit Text">
    Click on any text to edit the wording, add details, or fix AI errors.
  </Step>
  <Step title="Update Screenshots">
    Replace auto-selected screenshots with better frames from the video.
  </Step>
  <Step title="Add Sections">
    Insert additional steps or notes as needed.
  </Step>
  <Step title="Format Content">
    Use headers, bullet points, and highlights for better readability.
  </Step>
</Steps>

## Exporting Documents

<Tabs>
  <Tab title="Download">
    Click **Download** to export as:
    - PDF format
    - Markdown format
  </Tab>
  <Tab title="Share">
    Generate a shareable link for your documentation with:
    - Custom branding (Scale+ plans)
    - Viewer analytics
    - Password protection (optional)
  </Tab>
</Tabs>

## Document Templates

With **Pro plan and above**, you can create custom document templates in the [Brand Kit](/brand-kit/doc-templates):

- Custom headers and footers
- Brand colors and fonts
- Logo placement
- Standard formatting

## Multi-Language Support

Documents can be translated to any language:

1. Open the document
2. Click the **translation icon** in the toolbar
3. Select target language
4. AI generates translated version

<Tip>
  Translated documents maintain the same structure and screenshots - only text is translated.
</Tip>
