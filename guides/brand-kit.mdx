## Overview

The Brand Kit centralizes every asset that defines your organization's visual identity. Once configured, brand values propagate automatically to documents, [Shared Pages](/guides/shared-pages), and exports — ensuring consistent branding without per-document maintenance.

## Asset Categories

<CardGroup cols={2}>
  <Card title="Logos" icon="image">

  <Card title="Color Palette" icon="palette">

  <Card title="Typography" icon="font">

  </Card>
  <Card title="Voice & Tone" icon="comments">

  </Card>

## Configuration

<Frame caption="Brand Kit — configuration panel">
  ![Brand Kit scrolled](/images/0015_brand_kit_scrolled.png)
</Frame>

Brand Kit values are applied in the following contexts:

- Document and page default styling
- External [Shared Pages](/guides/shared-pages)
- PDF and slide exports
- Email notifications originating from the workspace
- Embedded widgets on external properties

## Governance

Brand Kit modification is typically restricted to Admin and Owner roles. Changes are versioned — prior configurations can be restored from the version history panel.
