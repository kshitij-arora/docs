## Overview

The Brand Kit is where every reusable branding asset lives. Once configured, these values propagate automatically into new videos and documents — no per-piece setup.

## Custom voices

Record or upload a ~5-minute sample to train a voice that sounds like you, or connect ElevenLabs via API key. See [Voices](/guides/voices).

## Custom avatars

Connect HeyGen with an API key, then pick any avatar from your HeyGen account inside Trupeer. See [Avatars](/guides/avatars).

## Custom backgrounds

Brand Kit → **Backgrounds** → **Upload an Asset** → pick a file. In any video, open **Visual → Background** to apply it.

## Custom music

Brand Kit → **Music** → **Add New Music** → name the track → upload the MP3 → **Add**. Appears in the music library for any video.

## Glossary — preserving brand terms

Define brand words (e.g., "ChatGPT", "YouTube") so the AI script and voice render them exactly as you specified, across languages and voices.

## Pronunciations

Add a new pronunciation with:

- **Phonetic text** (IPA) — e.g., `kriˈeɪʃən`
- **Recorded audio** — click the record button
- **Uploaded file** — MP3 or WAV

Preview, then **Add**. The pronunciation applies to future voiceover renders.

## Sync across videos

Changes to the Brand Kit apply to new videos and to any video that you re-render. Existing exported videos are not retroactively updated.
