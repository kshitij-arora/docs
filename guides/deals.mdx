## Overview

Deals is the opportunity-management surface of Trupeer. It models sales or partnership opportunities as records that progress through configurable stages, each with value estimates and probability-weighted forecasts.

## Deal Model

<CardGroup cols={2}>
  <Card title="Stage" icon="bars-progress">
    Current position in the pipeline — e.g., Discovery, Proposal, Negotiation, Closed.
  </Card>
  <Card title="Value" icon="dollar-sign">
    Expected contract value with optional currency and billing-cadence metadata.
  </Card>
  <Card title="Probability" icon="percent">
    Stage-derived or manually-set probability used to compute weighted pipeline value.
  </Card>
  <Card title="Primary Contact" icon="user-tie">
    Linked [Contact](/guides/contacts) record representing the deal's decision-maker or point of contact.
  </Card>
</CardGroup>

## Pipeline Views

- **Board** — Kanban-style columns organized by stage
- **List** — sortable table with configurable columns
- **Forecast** — period-over-period projection with probability-weighted totals

## Automation

Stage transitions can trigger automated actions — task creation, template-based messaging, or document generation. Automation rules are configured per-workspace under [Settings](/guides/settings) → Automations.
