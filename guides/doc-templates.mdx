## Overview

A document template captures typography, colors, logo placement, and the sections Trupeer should generate. It's applied to auto-generated docs so every new how-to looks and reads the same.

## Create a template

<Steps>
  <Step title="Open Brand Kit → Doc Templates">
    [Brand Kit](/guides/brand-kit) → **Doc Templates**. Pick **Set Manual Template** to build from scratch, or **Upload Existing Document** to derive a template from a sample.
  </Step>
  <Step title="Name the template">
    Give it a short identifier you'll recognize later.
  </Step>
  <Step title="Typography and color">
    Pick font family, body type, color, and default size. The same values apply to headings and body text.
  </Step>
  <Step title="Sections and prompts">
    Add named sections (Summary, Prerequisites, Steps, FAQ, etc.) with a prompt per section that tells the AI what to write there. The three-dot menu deletes or reorders a section.
  </Step>
  <Step title="Logo and branding">
    Upload a logo and pick its position.
  </Step>
  <Step title="Block quote styling">
    Toggle block quote styling on or off. When enabled, you can set its colors.
  </Step>
  <Step title="Save Changes">
    The template joins the **Doc Templates** list.
  </Step>
</Steps>

## Upload an existing doc to derive a template

If you already have an internal docs style, upload a representative sample. Trupeer analyses the structure and generates a matching template you can refine.

## Manage templates

Each template card's three-dot menu supports:

- **Rename**
- **Edit**
- **Set as Default**
- **Delete**

## Apply a template

Templates are applied at generation time (pick one on the **Generate AI Content** screen) or switched after the fact from the [Document Editor](/guides/documents) header.
