## Overview

Integrations expose Trupeer to the rest of your tool stack. Each integration is a bi-directional connector that can mirror records, trigger workflows, and surface external data within native Trupeer surfaces.

## Authentication Model

Integrations authenticate per-workspace and per-user. Workspace-level credentials enable shared access (e.g., a Slack workspace), while user-level credentials scope access to an individual's external account (e.g., a personal calendar).

## Webhook and API

For custom workflows, Trupeer exposes webhook endpoints for outbound events and a REST API for inbound operations. API credentials are managed under [Settings](/guides/settings) → Integrations.
