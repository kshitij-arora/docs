## Overview

The Knowledge Base is the canonical location for long-form, evergreen content. Unlike the [Library](/guides/library), which stores reusable assets, the Knowledge Base is optimized for authored documentation — articles, runbooks, policies, and FAQs.

## Reading and Authoring

<Frame caption="Knowledge Base — article detail view">
  ![Knowledge Base scrolled](/images/0016_knowledge_base_scrolled.png)
</Frame>

Articles are authored in a block-based editor that supports headings, callouts, tables, code blocks, and embedded media. Publication is controlled by a lightweight review workflow that can be disabled for workspaces that prefer unrestricted publishing.

## Search and Discovery

All Knowledge Base content is indexed by global [Search](/guides/search). Inline linking between articles — implicit via titles or explicit via link blocks — generates a knowledge graph visible from each article's "Related" section.
