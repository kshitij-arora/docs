## Overview

The Library is the unified asset catalog of your workspace. Every reusable artifact — documents, images, video, templates, and brand assets — is discoverable, versioned, and accessible from here.

## Organization Model

<CardGroup cols={2}>
  <Card title="Folders" icon="folder">
    Hierarchical grouping for coarse organization. Permissions can be scoped at the folder level.
  </Card>
  <Card title="Tags" icon="tags">
    Lightweight, multi-dimensional classification. An asset can carry any number of tags.
  </Card>
  <Card title="Collections" icon="layer-group">
    Saved views or playlists that group assets by use case rather than filesystem location.
  </Card>
  <Card title="Smart Filters" icon="filter">
    Dynamic groups defined by metadata queries — e.g., "All video assets tagged ‘onboarding' modified this quarter."
  </Card>
</CardGroup>

## Asset Operations

<Frame caption="Library — scrolled view with content">
  ![Library scrolled](/images/0014_library_scrolled.png)
</Frame>

Assets in the Library support the following operations:

- **Preview** — inline media playback and document previews without download
- **Version history** — every edit is retained with attribution and diff
- **Share** — generate scoped links with expiration and access controls
- **Transform** — format conversion and automated resizing for media
- **Embed** — reference assets from documents, pages, and [Shared Pages](/guides/shared-pages)
