## Overview

The Library is where every recording, generated video, and generated document lives. It defaults to a folder grid; each folder reports how many items it contains and whether it's empty.

## Per-item actions

Clicking into a folder exposes its videos and documents. Each item supports:

- **Open** — jumps to the [Video Editor](/guides/video-editor) or [Document Editor](/guides/documents)
- **Rename** — inline rename via the three-dot menu
- **Move** — relocate into a different folder
- **Delete** — send to trash (Admin-level policies may restrict this)
- **Share** — open the [Shared Pages](/guides/shared-pages) publish flow

## Item states

Generated content passes through visible states:

- **Queued** — in line for AI processing
- **Rendering** — video/document currently being generated
- **Draft** — ready to edit but not yet published
- **Published** — live on a Shared Page
