## Overview

Trupeer uses a persistent left sidebar as the primary wayfinding surface. Every destination in the product is reachable within two clicks from any screen.

<Frame caption="Primary navigation — sidebar expanded">
  ![Sidebar open](/images/0006_06_sidebar_open.png)
</Frame>

## Interface Regions

<CardGroup cols={2}>
  <Card title="Left Sidebar" icon="bars">
    Primary navigation. Houses top-level sections (Home, Library, Dashboard) and workspace switcher at the footer.
  </Card>
  <Card title="Top Bar" icon="magnifying-glass">
    Global search, quick-create action, notifications, and the active user menu.
  </Card>
  <Card title="Content Canvas" icon="window-maximize">
    The main working surface. Adapts its layout based on the active section.
  </Card>
  <Card title="Contextual Panels" icon="sidebar-flip">
    Right-aligned inspectors that surface metadata, collaborators, and inline actions for the selected object.
  </Card>
</CardGroup>

## Keyboard Efficiency

The sidebar can be toggled and content canvas can be traversed primarily via keyboard — the platform prioritizes low-latency input for power users.
