## Overview

Search is a first-class surface in Trupeer. It indexes every document, page, asset, contact, and message in your workspace and returns results ranked by recency, relevance, and the searcher's context.

## Capabilities

<CardGroup cols={2}>
  <Card title="Full-Text Indexing" icon="file-magnifying-glass">
    Body text, titles, captions, and metadata are indexed continuously. Changes appear in results within seconds.
  </Card>
  <Card title="Typed Filters" icon="filter">
    Narrow results by object type (pages, assets, contacts), author, date range, or workspace.
  </Card>
  <Card title="Contextual Ranking" icon="arrow-up-wide-short">
    Results incorporate your recent activity — content you've opened, edited, or shared ranks higher.
  </Card>
  <Card title="Keyboard-First" icon="keyboard">
    The search palette opens from anywhere in the product via keyboard shortcut for zero-friction lookup.
  </Card>
</CardGroup>

## Result Types

Search returns results across every object class in the platform:

- Documents and pages
- Assets in the Library
- Knowledge Base articles
- Contacts and team members
- Messages and comments
- Shared Pages
