## Overview

Tasks provides a lightweight but comprehensive task-tracking surface. Unlike standalone project management tools, Tasks is natively linked to workspace content — a task can reference a document, a contact, or a deal without leaving the platform.

## Views

Tasks can be browsed in multiple arrangements depending on workflow preference:

- **List** — flat, sortable view with inline editing
- **Board** — Kanban-style columns by status or assignee
- **Calendar** — due-date-driven timeline integrated with the [Calendar](/guides/calendar)
- **My Tasks** — personal filter showing only items assigned to the current user

## Integrations

Tasks created from third-party tools ([Integrations](/guides/integrations)) appear alongside native tasks with their origin attributed. Bi-directional sync keeps status changes consistent across systems.
