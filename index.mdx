## Introduction

Welcome to the Trupeer documentation. This site is the authoritative reference for every surface, workflow, and integration in the platform.

The documentation is organized into two complementary sections:

- **Guides** — conceptual walkthroughs of each surface, with illustrations and task-oriented examples
- **Reference** — structured, exhaustive reference material: URL map, permissions, shortcuts, and glossary

## Documentation Conventions

Throughout this documentation:

- **Mod** refers to **⌘** (Command) on macOS and **Ctrl** on Windows/Linux
- Code blocks in `monospace` indicate literal values, paths, or commands
- Screenshots reflect the current platform; minor visual differences may exist between releases
- Cross-references are linked — every guide and reference entry is reachable from related surfaces
