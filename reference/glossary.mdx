---
title: "Glossary"
description: "Canonical definitions for terms used throughout the Trupeer platform."
---

## A

**Admin**
Workspace role with full control — members, billing, integrations, and all content. See [Permissions & Roles](/reference/permissions).

**Annotation**
Text, shape, blur, or image overlay added to a video scene or a document screenshot. See [Annotations](/guides/annotations).

**Aspect Ratio**
Width-to-height ratio of the rendered video (16:9, 9:16, 1:1). Set per video or from a [Video Template](/guides/video-templates).

**Avatar**
An on-screen presenter generated from an AI avatar source (Trupeer presets or [HeyGen](/guides/avatars) clones). Optional per video.

## B

**Brand Kit**
Nine-tab reusable assets hub — templates, voices, avatars, backgrounds, logos, music, glossary, pronunciations. See [Brand Kit](/guides/brand-kit).

## C

**Caption**
On-screen subtitle generated from the script. Toggle per video or via template.

**Chapter**
A named segment of a video, used for navigation in the player and the Shared Page. See [Visuals](/guides/visuals).

**Custom Voice**
A voice model trained from a ~5-minute sample of the user's speech. See [Voices](/guides/voices).

## D

**Doc Template**
A saved documentation style — fonts, colors, logo, section prompts — applied when Trupeer generates a how-to. See [Doc Templates](/guides/doc-templates).

**Document**
The auto-generated how-to doc that accompanies every video. See [Documents](/guides/documents).

## E

**Editor**
Shorthand for either the [Video Editor](/guides/video-editor) or the [Document Editor](/guides/documents).

**ElevenLabs**
Third-party voice provider. Connect via API key for access to its voice catalog from inside Trupeer.

## G

**Glossary (Brand Kit)**
A list of brand terms with exact spellings, used to guide AI script generation.

## H

**HeyGen**
Third-party avatar provider. Connect via API key to use HeyGen-trained avatars inside Trupeer.

## I

**Intro / Outro**
Branded slides prepended or appended to a video. Configurable per video or in a Video Template.

## K

**Knowledge Base**
A branded help site assembled from Shared Pages. Add-on on Pro and above. See [Knowledge Base](/guides/knowledge-base).

## L

**Library**
Central catalog of your videos, documents, and folders. See [Library](/guides/library).

## P

**Pause**
A deliberate silence inserted in the script via the Script Editor. Preserved through voice regeneration.

**Pronunciation**
A phonetic override for a specific word. Stored in the Brand Kit; applied to voiceover renders.

## R

**Recording**
The Trupeer Chrome extension capture flow — screen, window, or tab with optional mic/camera. See [Recording](/guides/recording).

## S

**Scene**
A visually distinct segment of the rendered video — background, layout, and annotations. See [Visuals](/guides/visuals).

**SCIM**
System for Cross-domain Identity Management — enterprise user provisioning from an IdP.

**Script**
The textual source-of-truth for the video's voiceover. Editing the script triggers voice regeneration.

**Shared Page**
A video + document pair published at a `trupeer.ai/s/...` URL. See [Shared Pages](/guides/shared-pages).

**Skip**
A marked portion of the source recording that is excluded from the final render without deleting it — reversible.

**SSO**
Single Sign-On via SAML or OIDC. Enterprise-tier.

## T

**Translation**
A language-swapped copy of a video or document. See [Translations](/guides/translations).

**Trim**
Cropping start or end of the source video, or the rendered output. See [Script Editor](/guides/script-editor) and [Visuals](/guides/visuals).

## V

**Video Template**
A saved video style bundling voice, music, visuals, avatar, zoom, captions, and aspect. See [Video Templates](/guides/video-templates).

**Viewer**
Workspace role with read-only access. Cannot author.

## Z

**Zoom**
Auto-focus on the cursor or active region during a screen recording. Toggle per video or via template.
