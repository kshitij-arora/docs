<Note>
Shortcuts use **⌘** (macOS) and **Ctrl** (Windows/Linux). **Mod** refers to whichever is appropriate for the current platform.
</Note>

## Global

| Shortcut | Action |
|----------|--------|
| **Mod + K** | Command palette |
| **Mod + /** | Search the Library |
| **Mod + \\** | Toggle sidebar |
| **?** | Show all shortcuts |
| **Esc** | Dismiss modal or close panel |

## Navigation

| Shortcut | Action |
|----------|--------|
| **G then H** | Home |
| **G then L** | Library |
| **G then B** | Brand Kit |
| **G then S** | Shared Pages |
| **G then K** | Knowledge Base |
| **G then O** | Organization |

## Video Editor

| Shortcut | Action |
|----------|--------|
| **Space** | Play / pause |
| **← / →** | Seek 1 second |
| **Shift + ← / →** | Seek 5 seconds |
| **J / L** | Jump backward / forward |
| **I / O** | Mark in / out for trim |
| **Mod + S** | Save draft |
| **Mod + Z / Mod + Shift + Z** | Undo / Redo |

## Script Editor

| Shortcut | Action |
|----------|--------|
| **Mod + F** | Find |
| **Mod + Shift + H** | Find and Replace |
| **Mod + /** | Insert pause |
| **Mod + Enter** | Regenerate voiceover for selection |

## Document Editor

| Shortcut | Action |
|----------|--------|
| **Mod + B / I / U** | Bold / Italic / Underline |
| **Mod + K** then paste | Convert selection to link |
| **Mod + Shift + 7 / 8** | Ordered / Bullet list |
| **Mod + Shift + C** | Code block |
| **Mod + Z / Mod + Shift + Z** | Undo / Redo |

## Library

| Shortcut | Action |
|----------|--------|
| **Enter** | Open selected item |
| **V** | Toggle grid / title view |
| **Mod + Backspace** | Move to trash |
| **Mod + C** | Copy item link |
