## Philosophy

Trupeer is designed for keyboard-first operation. Every high-frequency action has a shortcut; shortcuts are consistent across surfaces.

<Note>
Shortcuts use **⌘** (Command) on macOS and **Ctrl** on Windows/Linux. Throughout this reference, **Mod** refers to whichever is appropriate for the current platform.
</Note>

## Global

| Shortcut | Action |
|----------|--------|
| **Mod + K** | Open command palette |
| **Mod + /** | Open global search |
| **Mod + \\** | Toggle sidebar |
| **Mod + ,** | Open Settings |
| **Mod + Shift + P** | Open profile |
| **?** | Show all shortcuts |
| **Esc** | Dismiss modal or close panel |

## Navigation

| Shortcut | Action |
|----------|--------|
| **G then H** | Go to Home |
| **G then D** | Go to Dashboard |
| **G then L** | Go to Library |
| **G then T** | Go to Tasks |
| **G then C** | Go to Calendar |
| **G then M** | Go to Messages |
| **G then S** | Go to Settings |

## Content Creation

| Shortcut | Action |
|----------|--------|
| **Mod + N** | New document |
| **Mod + Shift + N** | New folder |
| **Mod + D** | Duplicate current item |
| **Mod + Enter** | Submit form or send message |

## Content Editing

| Shortcut | Action |
|----------|--------|
| **Mod + B** | Bold |
| **Mod + I** | Italic |
| **Mod + U** | Underline |
| **Mod + Shift + X** | Strikethrough |
| **Mod + K** then paste | Convert selection to link |
| **Mod + Shift + 7** | Ordered list |
| **Mod + Shift + 8** | Bullet list |
| **Mod + Shift + 9** | Checklist |
| **Mod + Shift + C** | Code block |
| **Mod + Z** | Undo |
| **Mod + Shift + Z** | Redo |

## Messaging

| Shortcut | Action |
|----------|--------|
| **Mod + Shift + M** | Compose new message |
| **E** | Edit own message |
| **R** | Reply in thread |
| **Shift + R** | Reply inline |
| **↑** (empty composer) | Edit last message |

## Library and Assets

| Shortcut | Action |
|----------|--------|
| **Space** | Preview selected asset |
| **Enter** | Open selected asset |
| **Mod + C** | Copy asset link |
| **Mod + Backspace** | Move to trash |
| **V** | Toggle grid / list view |

## Tasks

| Shortcut | Action |
|----------|--------|
| **C** | Create task |
| **A** | Assign selected task |
| **D** | Set due date |
| **Shift + X** | Toggle complete |
| **Mod + ↑ / ↓** | Reorder task |

## Customization

Shortcuts can be remapped from [Profile](/guides/profile) → Keyboard. Workspace administrators can configure defaults under [Settings](/guides/settings) → Personalization.
