## Overview

Every screen in Trupeer is addressable by a stable URL. This map enumerates the canonical route for each surface and its corresponding guide.

## App routes

| Route | Surface | Guide |
|-------|---------|-------|
| `/` | Home — Start Recording, Upload, recents | [Home](/guides/home) |
| `/library` | Video and document library | [Library](/guides/library) |
| `/library/:folderId` | Folder contents | [Library](/guides/library) |
| `/videos/:id` | Video editor (default tab) | [Video Editor](/guides/video-editor) |
| `/videos/:id/document` | Document editor | [Documents](/guides/documents) |
| `/brand-kit` | Brand Kit — nine tabs | [Brand Kit](/guides/brand-kit) |
| `/shared-pages` | Published pages list | [Shared Pages](/guides/shared-pages) |
| `/knowledge-base` | Knowledge Base admin | [Knowledge Base](/guides/knowledge-base) |
| `/organization` | Members, seats, analytics | [Organization](/guides/organization) |
| `/organization/billing` | Plan and invoices | [Invoices](/guides/invoices) |
| `/pricing` | Plans and add-ons | [Pricing](/guides/pricing) |

## Public routes

| Route | Surface |
|-------|---------|
| `trupeer.ai/s/:slug` | Published Shared Page |
| `<custom-domain>/...` | Custom-domain Knowledge Base (Scale and above) |

## URL stability

Path segments above are stable; deep-link query parameters may change between releases.
