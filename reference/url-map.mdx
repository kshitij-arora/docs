## Overview

Every screen in Trupeer is addressable by a stable URL. This map enumerates the canonical route for each surface and its corresponding guide.

| Route | Surface | Guide |
|-------|---------|-------|

| `/settings` | Workspace configuration | [Settings](/guides/settings) |

All routes listed above are considered stable public-facing URLs. Deep-linked query parameters may change between releases; path segments remain backward-compatible.
