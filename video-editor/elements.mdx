## Elements Tab

Add overlay elements to highlight, annotate, or brand your videos.

## Available Elements

<CardGroup cols={3}>
  <Card title="Rectangle" icon="square">
    Add rectangular shapes for highlighting
  </Card>
  <Card title="Circle" icon="circle">
    Add circular shapes for emphasis
  </Card>
  <Card title="Blur" icon="eye-slash">
    Blur sensitive information
  </Card>
  <Card title="Text" icon="font">
    Add text annotations
  </Card>
  <Card title="Image" icon="image">
    Insert logos or graphics
  </Card>
</CardGroup>

## Adding Elements

<Steps>
  <Step title="Select Element Type">
    Click on the element you want to add (Rectangle, Circle, etc.)
  </Step>
  <Step title="Position on Timeline">
    Click **"+ Add elements"** in the timeline
  </Step>
  <Step title="Place on Video">
    Position the element in the video preview
  </Step>
  <Step title="Customize">
    Adjust size, color, and duration
  </Step>
</Steps>

## Element Properties

### Shapes (Rectangle, Circle)

| Property | Description |
|----------|-------------|
| **Color** | Fill color |
| **Opacity** | Transparency level |
| **Border** | Outline style |
| **Size** | Width and height |

### Blur

| Property | Description |
|----------|-------------|
| **Intensity** | How much to blur |
| **Area** | Region to blur |
| **Duration** | How long to show |

### Text

| Property | Description |
|----------|-------------|
| **Content** | Text to display |
| **Font** | Typography style |
| **Size** | Text size |
| **Color** | Text color |

### Image

| Property | Description |
|----------|-------------|
| **Source** | Upload or select image |
| **Size** | Dimensions |
| **Position** | Placement on screen |
