## Script Tab

The Script tab displays your AI-generated script organized by timestamp.

## Features

### Timestamped Segments

Your script is divided into segments with timestamps:

```
00:02  First segment of narration text here...
00:10  Second segment continues the explanation...
```

### Intro & Outro Scenes

Special markers indicate:

- **Intro Scene** - Opening title card (e.g., 0:00 - 0:02)
- **Outro Scene** - Closing card (e.g., 0:17 - 0:19)

## Editing Script

<Steps>
  <Step title="Click to Edit">
    Click on any text segment to edit the wording
  </Step>
  <Step title="Adjust Timing">
    Drag segment boundaries to change timing
  </Step>
  <Step title="Add Segments">
    Click **"+ Add"** to insert new script sections
  </Step>
  <Step title="Reorder">
    Drag segments to reorder content
  </Step>
</Steps>

## Script Actions

| Button | Function |
|--------|----------|
| **+ Add** | Add new script segment |
| **Regenerate** | Re-process AI script |
| **Settings** | Script formatting options |

## Syncing with Voiceover

After editing the script, you may need to re-sync the voiceover:

<Info>
  Click **"Refresh Voiceover"** to regenerate the AI voice narration with your edited script.
</Info>

## Adding Intro/Outro from Script

You can add intro and outro slides directly from the script section:

1. Click **+ Add** at the beginning for Intro
2. Click **+ Add** at the end for Outro
3. Select from templates or create custom
